Para aplicar o estilo interno eu posso aplicar a tag <style> dentro de cada corpo do codigo

Por exemplo:

<body style="background-color: COR;"> -> Comando usado para colocar cor no fundo do site

<body style="font-family: FONTE; font-size: 00px;"> -> Comando para mudar a fonte da letra do site e outro para alterar o tamanho

<h1 style="color: COR;"> -> Comando para colorir a fonte do H1 (ou qualquer outro)
