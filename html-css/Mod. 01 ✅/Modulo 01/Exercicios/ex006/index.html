Listas Ordernadas e Não Ordernadas

Listas Ordernadas

<ol>
<li> EX1 </li>
<li> EX2 </li>
</ol>

Algumas alterações da formatação dos tópicos <ol type = " " start = " "> </ol>

  • type: Tipo de formatação
  • start: Comando p/ saber em que posição começar a contagem na lista

  • Númerico (1)

    • Exemplo:

    1. EX1
    2. EX2
    3. EX3

  • Alfabeto Maiúsculo (A)

    • Exemplo:

    1. EX1
    2. EX2
    3. EX3

  • Alfabeto Minúsculo (a)

    • Exemplo:

    1. EX1
    2. EX2
    3. EX3

  • Númerico Romano (I)

    • Exemplo:

    1. EX1
    2. EX2
    3. EX3

  • Númerico Romano Minúsculo (i)

    • Exemplo:

    1. EX1
    2. EX2
    3. EX3

Listas Não Ordernadas

<ul>
<li> EX1 </li>
<li> EX2 </li>
</ul>

Algumas alterações da formatação dos tópicos <ul type = " " start = " "> </ul>

  • type: Tipo de formatação
  • start: Comando p/ saber em que posição começar a contagem na lista

  • Disco (disc) -> Padrão

    • Exemplo:

    • EX1
    • EX2
    • EX3

  • Circulo (circle)

    • Exemplo:

    • EX1
    • EX2
    • EX3

  • Quadrado (square)

    • Exemplo:

    • EX1
    • EX2
    • EX3

Ambas as listas citadas são importantes para a criação de menus

Listas Mistas e de definição

Listas Mistas

Pode-se misturas tanto as listas ordenadas (OL) quanto as Não Ordenadas (UL)

Lista de definição

<dl>
<dt>Termo</dt>
<dd>Descrição</dd>
</dl>

Exemplo:

HTML
Linguagem de marcação para criação do contéudo de um site.
CSS
Linguagem de marcação para criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
