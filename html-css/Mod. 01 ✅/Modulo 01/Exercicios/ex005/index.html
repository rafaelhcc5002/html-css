Formatações:

-Todas as formatações que eu saber, serão salvas aqui-

I. Negrito (Destaque)

Existem duas formas:

* <b>texto</b> (Este texto esta em negrito) → Tag B (Não semântica)
* <strong>texto</strong> (Este texto esta em negrito) → Tag Strong (semântica)

II. Italico (Ênfase)

Existem duas formas:

* <i>texto</i> (Este texto esta em italico) → Tag i (Não semântica)
* <em>texto</em> (Este texto esta em negrito) → Tag EM (semântica)

III. Texto marcado

*<mark>texto</mark> (Este texto esta marcado em amarelo) → Tag MARK

-Os proximos, usam um pouco de CSS:

* <mark style= "background-color:cor>texto marcado</mark> ( texto marcado com outra cor)
-Este acima, funciona exclusivamente para este texto que foi escrito ↑ ↑ ↑ ↑ ↑ ↑ ↑ ↑ ↑ ↑

IV. Texto grandes e pequenos

* <big>texto</big> (texto maior) → Tag BIG, para aumentar o texto -ESTE COMANDO ESTÁ OBSOLETO
* <small>texto</small> (texto menor) → Tag SMALL, para diminuir o texto

V. Texto deletado (Riscado)

* <del>texto</del> (texto) → Tag DEL

VI. Texto inserido (Sublinhado)

Existem duas formas:

* <ins>texto</ins> (texto) → Tag INS (semântica)
* <u>texto</u> (texto) → Tag U (Não semântica)

VII. Texto sobrescrito

* <sup>texto</sup> (x2+3) → Tag SUP (Em cima)

VIII. Texto subscrito

* <sub>texto</sub> (H2O) → Tag SUB (Em baixo)

IX. Codigo-fonte

São comandos, ou parte de um comandos
* <code>texto</code> (printf("%d textos lalala", numero);) → Tag CODE
-Esse codigo transforma em uma fonte monoespaçada

-Outra maneira, para se criar indetentações e espaçamentos é usando o CODE dentro do PRE:
