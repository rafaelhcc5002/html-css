Estilo CSS Interno

Coloca o CSS dentro do cabeçalho (head)

    <head>
        <style>
            body {
                background-color: COR;
            }
            h1 {
                color: COR;
            }
        </style>
    </head>

Estilo CSS Externo

Nesse, é necessário criar um arquivo externo chamado: style.css

No arquivo style.css coloque:

@charset "UTF-8;

EM seguida, pode colocar os parametros de estilo

Agora, no arquivo .html coloque no cabeçalho:

    <head>
        <link rel="stylesheet" href="style.css">
    </head>
