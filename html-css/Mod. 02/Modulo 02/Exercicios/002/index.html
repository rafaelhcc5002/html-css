Representando cores em CSS

  • Pode ser representado por nome: 

                            <style>

                            body { 
                                   color: COR;
                            }
                        </style>

  • Pode ser representado por código hexadecimal: 

                            <style>

                            body { 
                                   color: #CODIGOCOR;
                            }
                        </style>

  • Pode ser representado por código RGB: 

                            <style>

                            body { 
                                   color: rgb(VALOR, VALOR, VALOR);
                            }
                        </style>

  • Pode ser representado por código RGBA: 

                            <style>

                            body { 
                                   color: rgba(VALOR, VALOR, VALOR, VALOR);
                            }
                        </style>

  • Pode ser representado por código HSL: 

                            <style>

                            body { 
                                   color: hsl(VALOR, VALOR%, VALOR%);
                            }
                        </style>

  • Pode ser representado por código HSLA: 

                            <style>

                            body { 
                                   color: hsla(VALOR, VALOR%, VALOR%, VALOR);
                            }
                        </style>

Criar degradê em CSS

Para criar um degradê, você pode usar a propriedade background com a função linear-gradient.

Exemplo:

            <style>

                body { 
                    background: linear-gradient(DIREÇÃO, COR1, COR2);
                }
            </style>

Onde DIREÇÃO pode ser:

  • to right -> para direita
  • to left -> para esquerda
  • to top -> para cima
  • to bottom -> para baixo
  • to top right -> de cima pra direita
  • to top left -> de cima pra esquerda
  • to bottom right -> de baixo para direita
  • to bottom left -> de baixo para esquerda

Exemplo de uso:

            <style>

                body { 
                    background: linear-gradient(to right, red, blue);
                }
            </style>

Isso criará um degradê que vai do vermelho ao azul da esquerda para a direita.

Você também pode adicionar mais cores ao degradê, separando-as por vírgulas:

            <style>

                body { 
                    background: linear-gradient(to right, red, yellow, blue);
                }
            </style>

Isso criará um degradê que vai do vermelho ao amarelo e depois ao azul.

Você pode ajustar a opacidade das cores usando a função rgba:

            <style>

                body { 
                    background: linear-gradient(to right, rgba(255, 0, 0, 0.5), rgba(0, 0, 255, 0.5));
                }
            </style>

Isso criará um degradê que vai do vermelho semitransparente ao azul semitransparente.

Você pode usar a função radial-gradient para criar um degradê radial:

            <style>

                body { 
                    background: radial-gradient(circle, red, blue);
                }
            </style>

Isso criará um degradê radial que vai do vermelho ao azul.

Você pode ajustar a posição do centro do degradê radial usando a propriedade background-position:

            <style>

                body { 
                    background: radial-gradient(circle, red, blue);
                    background-position: 50% 50%;
                }
            </style>

Isso criará um degradê radial que vai do vermelho ao azul, com o centro do degradê na posição 50% 50%.

Você pode usar a função conic-gradient para criar um degradê cônico:

            <style>

                body { 
                    background: conic-gradient(red, blue);
                }
            </style>

Isso criará um degradê cônico que vai do vermelho ao azul.

Você pode ajustar a posição do centro do degradê cônico usando a propriedade background-position:

            <style>

                body { 
                    background: conic-gradient(red, blue);
                    background-position: 50% 50%;
                }
            </style>

Isso criará um degradê cônico que vai do vermelho ao azul, com o centro do degradê na posição 50% 50%.
