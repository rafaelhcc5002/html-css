↳ Recomendação

<a href = "LinkDoSite" target= "_blank" rel= " external"> </a>

Exemplo:

Na tag <target= " ">

  • _blank: Abre o link em uma nova aba em branco
  • _self (Padrão): Abre o link na mesma aba ou frame atual
  • _parent: Similar ao _top , em uma referência a janela mãe
  • _top: Vai desfazer todos os frames e abrir o destino no navegador completo

Na tag <rel=" ">

  • next: O link é para a próxima parte do documento atual
  • prev: O link é para a parte anterior do documento atual
  • external: O link é para outro site que não faz parte do site
  • nofollow: O link para um site não assegurado, como um link pago
  • author: O link para o site do autor do artigo atual
